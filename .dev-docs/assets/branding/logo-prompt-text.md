Create a premium modern logo for “SION MEDIA”, a multimedia worship platform for the Seventh-day Adventist Church.
Style: futuristic, elegant, glossy, cinematic, minimal yet luxurious, with strong 3D embossed effects and subtle neon glow.
Background: deep navy blue to black gradient with soft ambient lighting and vignette.

Main symbol requirements:

- Combine three symbolic elements into one unified icon:
  1. exactly THREE flame shapes inspired by the Adventist flame logo,
  2. a worship harp/lyre inspired by Lagu Sion,
  3. an open Bible/book at the bottom.

- The flames must be smooth, flowing, modern, and layered vertically.
- The harp should frame the flames elegantly with golden-orange gradients.
- The open Bible should glow subtly in blue and purple tones.
- The icon must feel balanced, symmetrical, premium, spiritual, and technological.

Typography:

- Large modern wordmark: “SION”
- Smaller subtitle below: “MEDIA”
- Futuristic sans-serif font, clean geometric style, metallic white/silver finish.
- Add subtle depth, reflections, shadows, and embossed 3D effect.
- Inside the letter “O” place a small play-button symbol representing multimedia/video projection.

Tagline:
“WORSHIP • WORD • MISSION”

- Use large centered separator dots (•) between words.
- Typography should look clean, professional, and well-spaced.

## Implementation Usage Note — 2026-05-07

Wordmark/logo presentation cocok untuk branding besar seperti splash, about screen, atau dokumentasi. Untuk title bar dan Program/Preview standby, gunakan versi transparent emblem dan render dengan mode contain agar tidak terpotong atau mendominasi UI operator.
