# UI Modernization Documents

This directory contains the UI redesign specifications for all three application modes.

## Documents

| File                        | Source                                           | Content                                                                                        | Authority |
| --------------------------- | ------------------------------------------------ | ---------------------------------------------------------------------------------------------- | --------- |
| `01-ui-system-parts1-6.md`  | `05-guides/phase3-ui-modernization-system-v1.md` | Global UI language, Title Bar, Library Mode, Projection Mode, Management Mode, Modal ecosystem | Level 2   |
| `02-ui-system-parts7-11.md` | `05-guides/phase3-part2-ui-parts7-11.md`         | Overlays, Projection visual system, Workflow UX, Page registry, Design-to-engineering handoff  | Level 2   |

## Reading Order

1. `01-ui-system-parts1-6.md` — understand the visual language and mode redesigns
2. `02-ui-system-parts7-11.md` — understand overlays, projection visuals, and workflow UX

## Key Information

- **01-ui-system-parts1-6.md** contains the complete visual specification for all three modes (Library, Projection, Management) including exact CSS values, layout dimensions, and interaction specifications.
- **02-ui-system-parts7-11.md** contains the projection visual system (typography, atmosphere, stage display), workflow UX modernization, and the complete page-by-page redesign registry.

## Implementation Note

UI modernization begins in Phase 5 (design system) and continues through Phase 8. Do not implement UI changes before the modal system (Phase 3) and projection hardening (Phase 4) are complete.
