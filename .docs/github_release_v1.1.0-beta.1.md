# SION Media v1.1.0-beta.1

> **Release Type:** Beta Testing  
> **Release Date:** 13 Juli 2026  
> **Platform:** Windows x64

Release beta ini memperkenalkan SION Link generasi baru, OBS Network Output berbasis SRT, impor PowerPoint dengan Speaker Notes, PowerPoint Presentation Bridge, perombakan Pengaturan Sistem, serta Stage Display yang lebih andal.

## Artefak Windows

- `SION-Media-1.1.0-beta.1-Setup.exe`
- `SION-Link-Desktop-1.1.0-beta.1-Setup.exe`

## Catatan beta

- Release belum merupakan versi stabil.
- Installer belum ditandatangani sertifikat code-signing publik sehingga Windows SmartScreen dapat menampilkan peringatan.
- Pengguna beta dianjurkan membuat backup data sebelum upgrade dan melaporkan masalah beserta Debug Report.

## Quality gate

- Lint dan TypeScript bersih.
- 65 test file dan 428 test SION Media lulus.
- Installer Windows kedua aplikasi berhasil dibangun dan diverifikasi checksum SHA-256.
